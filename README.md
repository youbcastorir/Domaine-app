## 📐 The Formula

```
P = ((K × B × S × E × R) / L^0.8) + log(A+1) × M
```

| Variable | Description |
|---|---|
| **P** | Final Domain Price (USD) |
| **K** | Keyword Strength (1–10) |
| **B** | Brandability Score (1–10) |
| **S** | Monthly Search Volume Score (1–10) |
| **E** | TLD Extension Multiplier (0.4–2.5) |
| **R** | Rarity Factor (1–10) |
| **L** | Domain Length (characters) |
| **A** | Domain Age (years) |
| **M** | Market Trend Multiplier (0.5–3.0) |
